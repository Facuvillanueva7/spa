# CV Interactivo con React

Single Page Application creada con React y Vite que presenta el currículum de Laura Martínez de forma interactiva.

## Scripts disponibles

- `npm install` — instala dependencias.
- `npm run dev` — levanta el entorno de desarrollo en `http://localhost:5173`.
- `npm run build` — genera la versión optimizada para producción.
- `npm run preview` — sirve la build de producción.

## Características destacadas

- Navegación entre secciones del CV en una sola página.
- Cambio de tema claro/oscuro y descarga rápida del perfil en texto.
- Filtros dinámicos de experiencia y detalle de habilidades por categoría.
